# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Context

This directory contains documentation for Claude Code installation rather than a development project. The primary content is installation instructions in Japanese for setting up Claude Code on Windows and Linux systems.

## Key Files

- `ClaudeCode構築.md` - Installation guide for Claude Code in Japanese, covering:
  - Windows installation via npm
  - PowerShell execution policy setup
  - Authentication and account linking process
  - Linux (Docker) installation notes

## Commands

Since this is a documentation repository without a development environment, there are no build, test, or lint commands. The main reference is the npm installation command:

```bash
npm install -g @anthropic-ai/claude-code@latest
```

## Architecture

This is not a code project but rather a documentation collection focused on Claude Code setup and configuration instructions.
